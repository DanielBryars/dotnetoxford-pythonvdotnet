# Python vs C# — Why Python Won ML

A Reveal.js presentation for .NET Oxford, March 2026.

## Viewing the slides

Open `index.html` in a browser, or serve it locally:

```bash
npx serve .
```

Keyboard shortcuts during the presentation:
- Arrow keys — navigate (right for next slide, down for sub-slides)
- `F` — fullscreen
- `S` — speaker notes
- `?print-pdf` — append to URL for PDF export

## Generating the plot images

The slides reference images generated by Python scripts. To create them:

```bash
cd examples/python
pip install -r requirements.txt
python lorenz.py       # generates images/lorenz.png
python ecosystem.py    # generates images/ecosystem.png
```

## Running the examples

### Python

```bash
cd examples/python
pip install -r requirements.txt
python autograd.py
python gpu.py
python mlp.py
python duck_typing.py
python lorenz.py
python ecosystem.py
python vectorisation.py
```

### C#

Each example is a separate project. Requires .NET 8+ SDK.

```bash
cd examples/csharp/Autograd && dotnet run
cd examples/csharp/Gpu && dotnet run
cd examples/csharp/Mlp && dotnet run
cd examples/csharp/DuckTyping && dotnet run
cd examples/csharp/Lorenz && dotnet run
```

The C# examples use TorchSharp — `dotnet restore` will pull the NuGet packages automatically on first run.

## Hosting on GitHub Pages

The slides use Reveal.js via CDN so there are no build steps. Push the repo and enable GitHub Pages (Settings > Pages > Source: main / root).
